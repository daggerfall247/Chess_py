Ein Schachspiel mit gerendertem Schachbrett. Man macht einen Zug, indem man erst auf eine Figur und anschließend auf das Zielfeld klickt.
Als Bibliothek wird pygame verwendet.

Die Anwendung befindet sich noch in Arbeit. Es spielt sich wie über einem echten Brett: Beide Spieler müssen selbst auf die Regeln achten.
D.h. es fehlen grundlegende Features wie Rochade, En passant und Beförderung. Außerdem fehlt eine Validierung der Züge und Zugreihenfolge.
