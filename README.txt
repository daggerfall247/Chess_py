Ein Schachspiel mit gerendertem Schachbrett. Man macht einen Zug, indem man erst auf eine Figur und anschließend auf das Zielfeld klickt.
Als Bibliothek wird pygame verwendet.

Die Anwendung befindet sich noch in Arbeit. Es fehlen grundlegende Features wie Rochade, En passant und Beförderung.
Außerdem fehlt eine Validierung der Züge.
